    **百分差异（percent difference）**，用于衡量两个数值的相对接近程度。计算公式如下：
    $$
    \mathrm{percent difference}=100\times\frac{|A-B|}{\frac{A+B}{2}}
    $$
4. 表格的简单操作   
    设表格数据如下：
    |Flavor|Color|Price|
    |:----:|:---:|:---:|
    |strawberry|pink|3.55|
    |chocolate|light brown|4.75|
    |chocolate|dark brown|5.25|
    |strawberry|pink|5.25|
    |chocolate|dark brown|5.25|
    |bubblegum|pink|4.75|

    <div className="code-compare">
    ```python
    import pandas as pd
    df = pd.read_csv("data.csv")
    df.head(2) # 显示前两行
    df['Flavor'] # 选择指定名称的列（不影响原表，下同）
    df[['Flavor', 'Price']] # 选择多列
    df.drop(columns = 'Color') # 拿走指定名称列
    df.sort_values(by = 'Price') # 按指定列排序（默认升序）
    df.sort_values(by = 'Price',ascending = False) # 降序排序
    df.loc[df['Flavor'] == 'chocolate'] # 筛选满足条件的行（注意大小写）
    ```

    ```r
    df <- read.csv("data.csv")
    head(df, 2)
    df$Flavor # 或 df[["Flavor"]]
    df[, c("Flavor", "Price")]
    df[, !names(df) %in% "Color"]
    df[order(df$Price), ]
    df[order(df$Price, decreasing = TRUE), ]
    df[df$Flavor == "chocolate", ]
    ```

    ```python
    import numpy as np
    baseline_high = 14.48
    highs = np.array([baseline_high - 0.880, baseline_high - 0.093,
                   baseline_high + 0.105, baseline_high + 0.684]) # 要求元素数据类型相同（若类型不同会强制转换）

    baseline_low = 3.00
    lows = np.array([baseline_low - 0.872, baseline_low - 0.629,
                    baseline_low - 0.126, baseline_low + 0.728])
    highs - lows # [11.472, 12.016, 11.711, 11.436]
    ```

    ```r
    baseline_high <- 14.48

          baseline_low - 0.126, baseline_low + 0.728)
    ```

2. 读取与处理表格数据
    <div className="code-compare">
    ```python
    import pandas as pd
    df = pd.read_table('data.csv', delimiter=',', index_col=false, nrows=5) # 取第一列为索引
    df.rename(columns={'a': 'A', 'b': 'B'}, inplace=False) # 修改指定列名（不覆盖原表）
    df.columns[1] # 获取列index=1的列名
    df.iloc[:,1] # 返回列index=1的数据
    df.iloc[:,1].iloc[2] # 返回列index=1,行index=2的数据
    df.shape # 获取dataframe大小
    styled_df = df.style.format({'Rate': '{:.2%}'}) # 转换为百分比
    df[df['A'].str.contains('C', na=False, regex=False)] # 筛选指定列中包含指定字符串的行（可设置正则表达式）
    ```

    ```r
    library(tidyverse)
    library(formattable)
    df <- read_csv('data.csv', n_max = 5)
    df_renamed <- df %>% rename(A = a, B = b)
    colnames(df)[2]   # 或 names(df)[2]
    df[[2]]           # 或 df[, 2, drop = TRUE]  
    df[[2]][3]        # 或 df[3, 2] 
    dim(df)        # 维数（行数为nrow(df)，列数为ncol(df)）       # 输出: 3
    styled_df <- df %>%
        formattable::formattable(list(Rate = percent))
    df %>%
        filter(str_detect(A, fixed("C")) & !is.na(A))
    ```
