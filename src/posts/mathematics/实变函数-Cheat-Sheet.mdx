# 集合论
## 集合及其运算
+ 基本概念（略，只列举一些新的符号定义）
    + 余集：设$\Omega$为全集，则$A$的余集定义为$\Omega\setminus A$，记为$A^c$；
    + 对称差：称集合$A\triangle B=(A\setminus B)\cup(B\setminus A)$为集合$A$与$B$的对称差；
    + 直积：$A\times B=\{(a,b):a\in A,b\in B\}$（也称笛卡尔乘积，注意顺序）
    
    相关性质：$A\setminus B=A\cap B^c$；$A\triangle B=(A\cup B)\setminus(A\cap B)$。
+ De Morgan公式：设$\Gamma$为一指标集，$\{A_t,t\in\Gamma\}$为任一集簇，则
    $$
    \left(\bigcup_{t\in\Gamma}A_t\right)^c=\bigcap_{t\in\Gamma}A_t^c,\quad\left(\bigcap_{t\in\Gamma}A_t\right)^c=\bigcup_{t\in\Gamma}A_t^c
    $$
### 集合的上、下极限集
+ 定义：设$\{A_n,n\geq 1\}$为任一集列，称集合$\{x:x$属于无穷多个$A_n\}$为集列$\{A_n,n\geq 1\}$的**上极限集**，记为$\displaystyle\varlimsup_{n\to\infty}A_n$；称集合$\{x:x$至多不属于有限个$A_n\}$为集列$\{A_n,n\geq 1\}$的**下极限集**，记为$\displaystyle\varliminf_{n\to\infty}A_n$.
+ 若$\displaystyle\varliminf_{n\to\infty}A_n=\varlimsup_{n\to\infty}A_n\triangleq A$，则称集列$\{A_n,n\geq 1\}$收敛，称$A$为其极限集，记为$\displaystyle\lim_{n\to\infty}A_n$.
+ 性质：
    + $\displaystyle\bigcap_{n=1}^{\infty}A_{n}\subset\varliminf_{n\to\infty}A_{n}\subset\varlimsup_{n\to\infty}A_{n}\subset\bigcup_{n=1}^{\infty}A_{n}$；
    + $\displaystyle\varlimsup_{n\to\infty}A_{n}=\bigcap_{n=1}^{\infty}\bigcup_{k=n}^{\infty}A_{k},\quad \varliminf_{n\to\infty}A_{n}=\bigcup_{n=1}^{\infty}\bigcap_{k=n}^{\infty}A_{k}.$
    > 第二条可以理解为“上极限集：对任意$n\geq 1$，存在$k\geq n$使其属于$A_k$”，“下极限集：存在$n\geq 1$，使得对任意$k\geq n$，其属于$A_k$”
+ 单调集列：
    + 单调不降：集列$\{A_n,n\geq 1\}$满足$A_1\subset A_2\subset\cdots\subset A_n\subset\cdots$；
    + 单调不增：集列$\{A_n,n\geq 1\}$满足$A_1\supset A_2\supset\cdots\supset A_n\supset\cdots$；
    + 二者合称单调集列。
    + 性质：单调集列必有极限集，且若集列单调不降，则$\displaystyle\lim_{n\to\infty}A_n=\bigcup_{n=1}^\infty A_n$；若集列单调不增，则$\displaystyle\lim_{n\to\infty}A_n=\bigcap_{n=1}^\infty A_n$.
### 示性函数
+ 定义：设集合$A\subset X$，称定义在$X$上的函数
    $$
    \chi_A(x)=\begin{cases}
    1,x\in A\\
    0,x\in X\setminus A
    \end{cases}
    $$
    为集合$A$的**示性函数**，也记为$I_A(x)$或$\mathbf{1}_A(x)$.

## 映射与基数
### 映射
+ 定义：设$X$与$Y$是两个非空集合，若存在一个法则$f$，使得对于$X$中的任一元素$x$，都有$Y$中的唯一确定的元素$y$与其对应，则称$f$是$X$到$Y$的**映射**，记为$f:X\to Y$.
    + 当映射$f$使得$y$与$x$对应时，称$y$为$x$在映射$f$下的**像**，记为$y=f(x)$；对于$y\in Y$，称满足$y=f(x)$的所有$x\in X$组成的集合为$y$在映射$f$下的**原像**，记为$f^{-1}(y)$.
        > $f^{-1}(y)$一般不是单元素集，且可能是空集；
+ 像集：对任意的$A\subset X$，记$f(A)=\{y\in Y:$存在$x\in A$使得$y= f(x)\}$为集合$A$在映射$f$下的**像集**；
    + 原像集：对于任意的$B\subset Y$，记$\displaystyle f^{-1}(B) = \bigcup_{y\in B}f^{-1}(y)$为集合$B$在映射$f$下的**原像集**;
+ 满射：若$f(X)=Y$，称$f$为$X$到$Y$的**满射**；
    + 单射：若对任意的$y\in Y$，除了$f^{-1}(y)=\emptyset$外，$f^{-1}(y)$皆为单元素集，称$f$为$X$到$Y$的**单射**；
    + 若$f$既是单射又是满射，称$f$为$X$到$Y$的**双射**或**一一映射**；
    + 逆映射：若$f$为$X$到$Y$的单射，则存在$f(X)$到$X$的映射使得对任一$y\in f(X)$，都有唯一的$x\in X$（即满足$f(x)=y$的$x$）与之对应，称这一$f(X)$到$X$的映射为$f$的**逆映射**，记为$f^{-1}:f(X)\to X$
        > 注意$f^{-1}$必为$f(X)$到$X$的双射.
    + 复合映射：设$f$为$X$到$Y$的映射，$g$为$f(X)$到$Z$的映射，则存在$X$到$Z$映射，使得对任一$x\in X$，都有唯一的$z=g(f(x))$与之对应。记这个映射为$g\circ f:X\to Z$，称其为映射$f$与$g$的**复合映射**。
### 对等与基数
+ 对等定义：若存在一个从集合$A$到集合$B$的一一映射，则称$A$与$B$**对等**，记为$A\sim B$.
    + 对等关系属于等价关系，具有自反性、对称性与传递性。
+ **Bernstein定理**：若集合$A$与$B$的某一子集对等，$B$与$A$的某一子集对等，则$A\sim B$.
    + 特例：若集合$A\subset C$，且$C\subset B\subset A$，则$A\sim B$。
    > 设$-\infty\leq a< b\leq\infty$，则$(a,b)\sim(-\infty,\infty)$.
+ 基数定义：**基数**用于刻画相互对等的集合的某种共同属性，用$\overline{\overline{A}}$表示。其满足以下性质：
    + 若$A\sim B$，则$\overline{\overline{A}}=\overline{\overline{B}}$；
    + 若存在$C\subset B$使得$A\sim C$（或存在$A$到$B$的一个单射），称$A$的基数不超过$B$的基数，记为$\overline{\overline{A}}\leq\overline{\overline{B}}$或$\overline{\overline{B}}\geq\overline{\overline{A}}$。
        + 若$\overline{\overline{A}}\leq\overline{\overline{B}}$且$\overline{\overline{A}}\neq\overline{\overline{B}}$，则称$A$的基数小于$B$的基数，记为$\overline{\overline{A}}< \overline{\overline{B}}$.
        + 这样Bernstein定理也可表述为：若$\overline{\overline{A}}\leq\overline{\overline{B}},\overline{\overline{B}}\leq\overline{\overline{A}}$则$\overline{\overline{A}}=\overline{\overline{B}}$.
### 可数集
+ 有限集与无限集：若存在正整数$n$,使得集合$A$与集合$\{1,2,\cdots,n\}$对等，称$A$为**有限集**，且$\overline{\overline{A}}=n$；称不是有限集的集合为**无限集**。
    + 约定$\emptyset$为有限集，且$\overline{\overline{\emptyset}}=0$.
+ 可列集：若集合$A$与正整数集$\{1,2,\cdots\}$对等，称$A$为可列集，其基数记为$\aleph_0$（读作“阿列夫零”）.
+ 有限集与可列集统称为**可数集**。
+ 性质：
    + 任何无限集都包含一个可列子集；（可列集是基数最小的无限集）
    + 可列集的任何无限子集必为可列集；
    + 可数个可数集的交、并、差等运算后的结果仍为可数集；特别地，可数个可列集的并必为可列集。
    + 有理数集$\mathbb{Q}$是可列集；
    + 设集合$A$是可数集，$B$是无限集，则$\overline{\overline{A\cup B }}=\overline{\overline{B}}$，即无限集并上一个可数集后基数不发生改变；
    + 集合$A$为无限集的充分必要条件为$A$与其某个真子集对等.
### 不可数集
+ $(0,1)=\{x:0< x< 1\}$不是可数集。称$[0,1]$的基数为**连续基数**，记为$c$或$\aleph$；
    + $(a,b),[a,b),(-\infty,\infty)$均具有连续基数。
+ 无最大基数定理：设$A\neq\emptyset$，$\mathcal{P}(A)$表示$A$的幂集，则$A$与$\mathcal{P}(A)$不对等，且$\overline{\overline{A}}< \overline{\overline{\mathcal{P}(A)}}$
+ 连续统假设：不存在集合$A$，使$\aleph_0< \overline{\overline{A}}< c$.
## 开集
### 欧氏空间
+ 设$X$是一个集合，任意$x,y\in X$,有实数$d(x,y)$与之对应，具有性质：
    + $d(x,y)\geq 0$，取等号当且仅当$x=y$；
    + $d(x,y)=d(y,x)$；
    + 对任意$z\in X$，$d(x,y)\leq d(x,z)+d(z,y)$.
    
    则称$d(\cdot,\cdot)$为定义在$X$上的**距离**，称序对$(X,d)$为**距离空间**。
+ 设$x=(x_1,\cdots,x_n)\in\R^n$，称$\displaystyle |x|=\sqrt{\sum_{k=1}^n x_k^2}$为$x$的**范数**或**长度**。
    + 称距离空间$(\R^n,d)$为$n$维**欧氏空间**，其中$d(x,y) =|x - y|$.（也可直接用$\R^n$表示欧氏空间）
+ 有界集与无界集：设$E\subset\R^n$，称
    $$
    \mathrm{diam}E=\sup\{|x-y|:x,y\in E\}
    $$
    为点集$E$的直径。若$\mathrm{diam}E< \infty$，称$E$为**有界集**，否则称$E$为无界集。
    + 非空集合$E\subset\R^n$有界当且仅当存在$M>0$，使得对任意的$x\in E$，都有$|x|\leq M$.
+ 对于$\mathbb{R}^n$的点列$\{x_n, n \geq 1\}$，若存在$x \in \mathbb{R}^n$，使得$\displaystyle\lim_{n \to \infty} |x_n - x| = 0$，则称$x$为点列$\{x_n, n \geq 1\}$的**极限**，记为$\displaystyle\lim_{n \to \infty} x_n = x$,
并称$\{x_n, n \geq 1\}$为**收敛点列**。
### 开集与闭集
+ 开集定义：设$x\in\R^n,A\subset\R^n$，
    + 对任意$\epsilon>0$，称集合$\{U(x,\epsilon)\}=\{y\in\R^n:d(x,y)< \epsilon\}$为点$x$的$\epsilon$-**邻域**；
    + 若存在邻域$U(x,\epsilon)\subset A$，则称$x$为$A$的一个**内点**；集合$A$的所有内点构成的集合称为$A$的**内部**，记为$A^\circ$；
    + 若集合$A$满足$A\subset A^\circ$，称$A$为$\R^n$中的**开集**。规定$\emptyset$为开集。
+ 开集性质：
    + $\emptyset,\R^n$都是开集；任意多个开集的并还是开集；有限多个开集的交是开集。
+ 闭集定义：设$x \in \mathbb{R}^n, A \subset \mathbb{R}^n$，
    + 任意的$\varepsilon > 0$，邻域$U(x, \varepsilon)$包含$A$的无限多个点，则称$x$为$A$的一个**聚点**或**极限点**。集合$A$的所有聚点构成的集合称为$A$的**导集**，记为$A'$。
    > 有限集的导集为空集；集合$A$的聚点未必属于$A$
    + 若集合$A$满足$A'\subset A$，则称$A$为$\mathbb{R}^n$中的**闭集**。称$\overline{A} = A \cup A'$为集合$A$的**闭包**。
+ 导集与闭包的性质：
    + 设$A \subset \mathbb{R}^n$，则  
        + $x \in A'\Longleftrightarrow$存在点列$\{x_n, n \geq 1\} \subset A \setminus \{x\}$，使得当$n \to \infty$时，$x_n \to x$； 
        + $x \in \overline{A}\Longleftrightarrow$存在$A$中的点列$\{x_n, n \geq 1\} \subset A$，使得当$n \to \infty$时，$x_n \to x$.
    + 若$A, B \subset \mathbb{R}^n$，则$(A \cup B)' = A' \cup B'$；
    + 设$A \subset \mathbb{R}^n$，则$A', \bar{A}$皆为闭集；
    + 若$\{A_n, n \geq 1\} \subset \mathbb{R}^n$，则$\displaystyle\left( \bigcup_{n=1}^\infty A_n \right)' \subset \bigcup_{n=1}^\infty A'_n$未必成立。（反例：$\{A_n\}$为所有有理数集合）
+ 闭集性质：
    + $\emptyset$和$\R^n$都是闭集；任意多个闭集的交还是闭集；有限多个闭集的并是闭集。
    > $\R^n$中既是开集又是闭集的集合只有$\emptyset$和$\R^n$。
    + 若$A\subset\R^n$为非空集合，$x\in\R^n$，定义$d(x,A)= \inf\{d(x,y):y\in A\}$。则$A$为闭集$\Longleftrightarrow$对任意$x\in\R^n$，存在$y\in A$，使得$d(x,y)=d(x,A)$.
    + 设$A\subset\R^n$，则$A$是闭集$\Longleftrightarrow$A中的任意收敛点列的极限属于$A$.
+ 闭集与开集的关系：设$A\subset\R^n$，则$A$为闭集$\Longleftrightarrow A^c$为开集。
+ 稠密性：设$A$和$B$都是$\R^n$的子集，若$\overline{A}\supset B$，则称$A$在$B$中**稠密**；
    + 特别地，若还有$A\subset B$，则称$A$是$B$的**稠密子集**；若$(\overline{A})^\circ=\emptyset$，则称$A$为**疏朗集**或**无处稠密集**。
        > 有理数集$\mathbb{Q}$是$\R^1$的稠密子集；有理数集$\mathbb{Q}$在无理数集$\R^1\setminus Q$中稠密；整数集$\Z$是疏朗集。
### 开集的构造与Borel集
> 将集合$A$的所有开子集组成的集类记为$\mathcal{O}(A)$.
+ 构成区间：设集合$G \in \mathcal{O}(\mathbb{R}^1), \alpha, \beta \notin G$，但$(\alpha, \beta) \subset G$，称这样的区间$(\alpha, \beta)$为开集$G$的**构成区间**。
+ 定理：$\mathbb{R}^1$中的非空开集必是可数个互不相交的开区间的并；$\mathbb{R}^1$中的闭集必是从$\mathbb{R}^1$中挖去可数个互不相交的非空开区间所得的集合。
+ $\sigma$-代数：定义略（见概率论）。
+ Borel $\sigma$-代数：称$\sigma(\mathcal{O}(\R^n))$为**Borel $\sigma$-代数**，记为$\mathcal{B}(\R^n)$或$\mathcal{B}$。$\mathcal{B}$中的元素称为**Borel集**。
+ 若$E\subset\R^n$是可列个开集的交集，则称$E$为$G_\delta$型集；若$E$是可列个闭集的并集，则称$E$是$F_\sigma$型集。
### 连续函数
+ 定义：设$E \subset \mathbb{R}^n$，$f$是定义在$E$上的实值函数。$x_0 \in E$，若对任意的$\varepsilon > 0$，存在$\delta > 0$，使得$f(U(x_0, \delta) \cap E) \subset U(f(x_0), \varepsilon)$，则称$f$在$x_0$处**连续**。
    + 若$f$在$E$中的每一点处都连续，则称$f$为$E$上的**连续函数**。$E$上的连续函数全体记为$\mathcal{C}(E)$。
+ 连续函数的延拓：若$F \subset \mathbb{R}^n$为闭集，$f \in\mathcal{C}(F)$，且存在$M > 0$，使得对任意的$x \in F$，都有$|f(x)| \leq M$成立，则存在函数$g \in\mathcal{C}(\mathbb{R}^n)$，使得对任意的$x \in \mathbb{R}^n$，都有$|g(x)| \leq M$成立。
    + 特别地，当$x \in F$时，$g(x) = f(x)$。
## Cantor集
+ Cantor集的构造：将$[0,1]\subset\R^1$三等分，去掉中间开区间$\displaystyle\left(\frac{1}{3},\frac{2}{3}\right)$，记剩下部分为$\displaystyle F_1=\left[0,\frac{1}{3}\right]\cup\left[\frac{2}{3},1\right]=F_{1,1}\cup F_{1,2}$；再对$F_{1,1},F_{1,2}$三等分，取左右两部分，记为$F_2=F_{2,1}\cup F_{2,2}\cup F_{2,3} \cup F_{2,4}$；以此类推，得到集列$\{F_n,n\geq 1\}$，其中
    $$
    F_n=\bigcup_{i=1}^{2^n}F_{n,i},\quad n=1,2,\cdots
    $$
    称集合$\displaystyle C=\bigcap_{n=1}^\infty F_n$为**Cantor（三分）集**。
+ 性质：
    + $C$为非空有界闭集；
    + $C=C'$，即$C$为完全集（没有孤立点）；
    + $C$没有内点；
    + $C$是疏朗集；
    + $C$具有连续基数$c$，即$\overline{\overline{C}}=c$；
    + 被挖去的小区间长度和为$1$.
# 测度论
+ 定义广义实数集$\overline{\R}^1=\R^1\cup\{-\infty,\infty\}$，规定以下计算规则（设$a\in\R^1$）：
    + $a\pm\infty=\pm\infty,+\infty+(+\infty)=+\infty,-\infty-(+\infty)=-\infty$；
    + $(\pm\infty)\cdot(\pm\infty)=+\infty,(+\infty)\cdot(-\infty)=-\infty,\dfrac{a}{\pm\infty}=0$；
    + $$
        a\cdot(\pm\infty)=\begin{cases}
            \pm\infty,&a>0\\
            0,&a=0\\
            \mp\infty,&a<0
        \end{cases}
      $$
    > 特别声明$+\infty-(+\infty)$没有意义.
## 外测度
+ **开矩体**：称集合$I =\{(x_1,x_2,\cdots ,x_n)\in\R^n: a_k< x_k< b_k, a_k,b_k\in\R,a_k< b_k,k=1,2,\cdots,n\}$为$\R^n$中的一个开矩体。规定$I$的长度为
    $$
    |I|=\prod_{k=1}^n(b_k-a_k)
    $$
    > 类似可以定义$\R^n$中的**闭矩体**、**半开闭矩体**等，其长度均定义为边长乘积。
+ **L-覆盖**：设集合$E\subset\R^n$，若可数个开矩体$I_1,I_2,\cdots$，使得$\displaystyle\bigcup_{k=1}^\infty I_k\supset E$，则称$\{I_1，I_2,\cdots\}$为$E$的一个L-覆盖。
    + 性质：
    1. 若$E\subset F$,则$F$的L-覆盖必是$E$的L-覆盖；
    2. 记$I_k=(-k,k),J_k=(-2k,2k),k=1,2,\cdots$,则$\{I_k,k\geq 1\}$和$\{J_k,k\geq 1\}$都是$\R$的L-覆盖；
    3. 任何集合$E\subset\R^n$都存在一个L-覆盖，且不唯一.
+ （Lebesgue）外测度定义：设集合$E\subset\R^n$，称
    $$
    m^*(E)=\inf\left\{\sum_{k=1}^\infty|I_k|:\{I_k,k\geq 1\}\text{为}E\text{的一个L-覆盖}\right\}
    $$
    为$E$的**Lebesgue外测度**。
    + 任何集合$E\subset\R^n$的外测度$m^*(E)$均存在；
    + 单点集的外测度为$0$。
+ 称外测度为$0$的集合为**零测集**。
    + Cantor集，可数集均为零测集。
+ 设$I$为开矩体，则$m^*(I)=|I|$.
+ 性质：
    1. **非负性**：任意$E \subset \mathbb{R}^n$，都有$m^*(E) \geq 0$；
    2. **单调性**：若$E_1 \subset E_2$，则$m^*(E_1) \leq m^*(E_2)$；
    3. **次可加性**：$\displaystyle m^* \left( \bigcup_{k=1}^{\infty} E_k \right) \leq \sum_{k=1}^{\infty} m^*(E_k)$.
+ 外测度不具有可加性，需要限制条件：
    + 若$A, B \subset \mathbb{R}^n,d(A, B) = \inf\{d(x, y) : x \in A, y \in B\} > 0$，则有 
        $$
        m^*(A \cup B) = m^*(A) + m^*(B).
        $$
## 测度
> 测度的引入目的是为了解决外测度的不可加性问题
+ 定义（Carathéodory条件，简称**卡氏条件**）：设$E\subset\R^n$，若对任意的$T\subset\R^n$（也称为**试验集**），都有
    $$
    m^*(T)=m^*(T\cap E)+m^*(T\cap E^c)
    $$
    成立，则称$E$为Lebesgue可测集，简称$E$为**可测集**或$E$可测，记为$E\in\mathcal{M}$（$\mathcal{M}$表示所有可测集组成的全体）；$m^*(E)$称为$E$的**测度**，记为$m(E)$.
    + 零测集必可测，且测度为$0$；
    + 由外测度的次可加性，卡氏条件可化为$m^*(T)\geq m^*(T\cap E)+m^*(T\cap E^c)$
+ 等价条件：$E\in\mathcal{M}\Longleftrightarrow$任意$A\subset E,B\subset E^c$，有$m^*(A\cup B)=m^*(A)+m^*(B)$
    + 推论：若$E_1,E_2\in\mathcal{M}$，且$E_1\cap E_2=\emptyset$，则对任意的$T\subset\R^n$，都有
        $$
        m^*(T\cap(E_1\cup E_2))=m^*(T\cap E_1)+m^*(T\cap E_2)
        $$
        特别地，令$T=\R^n$，则
        $$
        m^*(E_1+E_2)=m^*(E_1)+m^*(E_2)
        $$
        （即推出测度的有限可加性）
        > 在上面的推论中，可以不要求$E_2$可测（因为$(T\cup E_2)\subset E_2\subset E_1^c$）
+ 卡氏条件简化版本：$E\in\mathcal{M}\Longleftrightarrow$对任意的开矩体$I$，有
    $$
    |I|\geq m^*(I\cap E)+m^*(I\cap E^c).
    $$
### 可测集的性质
1. $E \in \mathcal{M}\Longleftrightarrow E^c \in \mathcal{M}$;
2. 若$E_1, E_2 \in \mathcal{M}$，则$E_1 \cup E_2, E_1 \cap E_2$和$E_1 \setminus E_2$都在$\mathcal{M}$中;
3. 若$E_1, E_2, \dots \in \mathcal{M}$，且当$i \neq j$时，还有$E_i \cap E_j = \emptyset$，则$\displaystyle\sum_{k=1}^\infty E_k \in \mathcal{M}$，且
    $$
    m\left(\sum_{k=1}^\infty E_k\right) = \sum_{k=1}^\infty m(E_k)
    $$
    （测度可列可加性）
4. 若$E_1, E_2, \dots \in \mathcal{M}$，则$\bigcup_{k=1}^\infty E_k \in \mathcal{M}$.
### 测度的性质
1. $m(\emptyset) = 0$;
2. $\displaystyle m\left(\sum_{k=1}^{\infty} A_k\right) = \sum_{k=1}^{\infty} m(A_k)$；
3. $m(A + B) = m(A) + m(B)$；
4. 若 $A \subset B$，则 $m(A) \leq m(B)$；
5. 若 $A \subset B$, $m(A) < \infty$，则 $m(B \setminus A) = m(B) - m(A)$；
6. 若 $m(AB) < \infty$，则 $m(B \setminus A) = m(B) - m(AB)$；
7. $\displaystyle m\left(\bigcup_{k=1}^{\infty} E_k\right) \leq \sum_{k=1}^{\infty} m(E_k)$
+ 测度的上、下连续性：设$\{E_n,n\geq 1\}\in\mathcal{M}$，
    + **测度的下连续性**：若$E_1 \subset E_2 \subset \cdots$，则$\displaystyle\lim_{k \to \infty} m(E_k) = \lim_{k \to \infty} m(E_k)$；
    + **测度的上连续性**：若$m(E_1) < \infty, \, E_1 \supset E_2 \supset \cdots$，则$\displaystyle\lim_{k \to \infty} m(E_k) = \lim_{k \to \infty} m(E_k)$。
## 可测集与开集
+ $\R^n$中的任意开矩体都是可测集$\Longrightarrow\R^n$中的任意开集都是可测集
    + 闭集为可测集；$G_\delta$型集和$F_\sigma$型集都是可测集；Borel集是可测集。
+ 可测集的特征：
    + 对任意的$\epsilon>0$，存在开集$G$和闭集$F$，使得$F\subset E\subset G$，且
        $$
        m(G\setminus E)< \epsilon,\quad m(E\setminus F)< \epsilon
        $$
    + 存在$G_\delta$型集$G^*$和$F_\sigma$型集$F^*$。使得$G^*\supset E\supset F^*$且
        $$
        m(G^*\setminus E) = m(E\setminus F^*)=0
        $$
    > 由此可见，可测集“差不多”就是开（闭）集。
## 不可测集
+ 平移：设集合$A\subset\R^1$ ，称集合$a+A=\{a+x:x\in A\}$为$A$的一个**平移**。
+ 定理：设$a \in \mathbb{R}^1$。
    1. 对任意的$A \subset \mathbb{R}^1$，有$A = a + ((-a) + A)$。
    2. 设$I$ 是开区间，则$a + I$ 也是开区间，且$|a + I| = |I|$；
    3. 设$A \subset \mathbb{R}^1$，则$m^*(a + A) = m^*(A)$；
    4. 若$A \in \mathcal{M}$，则$a + A \in \mathcal{M}$，且$m(a + A) = m(A)$。
+ 在$[0,1]$中定义等价关系$\sim$，即若$x,y\in[0,1]$且$x-y\in\mathbb{Q}$，则$x\sim y$。
    + 由此，$[0,1]$被分为一些等价类集合，由选择公理，我们可以在每个等价类中选择且仅选择一个元素，这样构成的集合记为$E$。则$E\subset [0,1]$且$E$是Lebesgue不可测集。
    + 推论：存在两个不相交的实数集$A$和$B$，使得
        $$
        m^*(A\cup B) < m^*(A) + m^*(B).
        $$
